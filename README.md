## Project Overview

This repository contains the technical component of the Capstone Project for the OSS course. The project involves a structured audit of the **Linux Kernel**, focusing on its origin, GPL v2 licensing, and its functional footprint within a Linux ecosystem.

The audit is grounded in the philosophy that open source software — built openly, shared freely, and improved collectively — represents one of the most significant movements in modern computing.

## How to Run

**1. Clone the repository:**

```bash
git clone https://github.com/BiswarupMukherjee2005/Open-Source-Audit.git
cd Open-Source-Audit
```

**2. Grant execution permissions:**

```bash
chmod +x *.sh
```

**3. Execute a script:**

```bash
# Script 1 — System Identity
./system_identity.sh

# Script 2 — Package Inspector
./package_inspector.sh

# Script 3 — Directory Auditor (may require sudo)
sudo ./directory_auditor.sh

# Script 4 — Kernel Log Analyzer
./kernel_log_analyzer.sh /var/log/kern.log

# Script 5 — Manifesto Generator
./manifesto_generator.sh
```

## Philosophical Note

> *"Free software is a matter of liberty, not price."* — Richard Stallman

By auditing the Linux Kernel, this project highlights how the **GPL v2 license** protects the four essential freedoms of software:

- **Freedom 0** — The freedom to run the program for any purpose
- **Freedom 1** — The freedom to study how the program works
- **Freedom 2** — The freedom to redistribute copies
- **Freedom 3** — The freedom to distribute modified versions

The Linux Kernel is a living testament to what becomes possible when these freedoms are upheld — a project started by a single developer in 1991 that now powers everything from Android smartphones to the world's top supercomputers.
